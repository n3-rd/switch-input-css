Switch Input CSS

Making your check and radio boxes better with IOS-styled buttons

What Is Switch-input.css?

Switch input css is a css library that enables you to add ios-like checkboxes and radio inputs to your code.

It comes in 5 colours

<input type="checkbox"><span>Default</span>
